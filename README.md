## 📖 Описание

Это мой **второй полноценный проект** — веб-приложение с регистрацией, входом и личным кабинетом.  
Данные хранятся в браузере (`localStorage`), что имитирует работу с базой данных.

Проект показывает, как:
- строить многостраничные сайты
- работать с формами
- сохранять данные пользователя
- делать красивые анимации и размытый фон

## 🚀 Живая демка

👉 [Открыть сайт](https://eridan-forge.github.io/project-2-auth-app)

## 🎨 Функциональность

- ✅ Главная страница с анимированными карточками
- ✅ Форма регистрации (имя, email, пароль)
- ✅ Личный кабинет с приветствием и списком проектов
- ✅ Кнопка «Выйти» — очищает данные
- ✅ Размытый фоновый изображение (`backdrop-filter`)
- ✅ Адаптивная вёрстка (работает на телефоне)
